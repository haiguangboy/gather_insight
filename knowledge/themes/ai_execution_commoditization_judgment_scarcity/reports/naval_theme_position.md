# Naval Theme Position

This file separates Naval's direct source positions from system synthesis and guest engineering evidence.

## Core logic chain

1. AI and agents lower implementation and iteration cost.
2. Output volume and leverage increase, but problem selection and judgment remain uneven.
3. Cheap execution moves bottlenecks toward architecture, verification, real-world constraints, and institutional permission.
4. High-leverage individuals and smaller teams may create more firms rather than simply eliminating all work.
